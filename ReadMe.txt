🔋 최종 발전량 예측 모델
이 프로젝트는 태양광 및 풍력 발전량 예측 모델을 통합하여, 사용자 입력에 따라 원하는 발전원을 선택해 예측을 수행하는 Python 기반 콘솔 애플리케이션입니다.
📁 최종 실행 파일

최종_발전량_예측_모델/
└── code/
    └── run_prediction.py   ← 해당 파일을 실행하면 전체 예측이 수행됩니다.
🧠 작동 방식
run_prediction.py는 다음과 같은 흐름으로 작동합니다:
    1. 태양광 또는 풍력 발전 예측 모듈을 동적으로 불러옴
    2. 사용자가 실행할 예측 종류(태양광 / 풍력)를 선택
    3. 선택한 모델의 main() 함수를 실행하여 예측 수행
    4. 예측 결과는 .csv 파일로 저장됨

모델 경로는 상위폴더의
태양광_발전량_예측_모델/2. 최종 예측 모델/코드/solar_final_model.py,
풍력_발전량_예측_모델/3. 최종 예측 모델/코드/wind_final_model.py에 위치한 스크립트를 불러옵니다. 

🛠 실행 방법
최종 실행 파이썬 파일을 run합니다.

실행 후 콘솔에 나타나는 메뉴를 통해 예측을 진행할 수 있습니다.

📦 예측 결과
예측이 완료되면 각 모델은 결과를 아래 경로에 CSV 파일로 저장합니다:

최종_발전량_예측_모델/
└── result/

    solar_predictioin_results.csv: 태양광 발전량 예측 결과

    wind_prediction_results.csv: 풍력 발전량 예측 결과

⚠️ 유의사항
모델 파일이 지정된 상대경로에 존재해야 합니다.

실행 전 필요한 라이브러리가 설치되어 있어야 합니다 (pandas, numpy, 등).

VS Code 등의 IDE에서는 전체 프로젝트 폴더를 오픈한 상태에서 실행해야 정상적으로 작동합니다.
